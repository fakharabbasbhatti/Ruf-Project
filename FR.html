About This Project

Technology Used

This project is built with pure HTML5, CSS3, and JavaScript. It uses Bootstrap for responsive layout and Font Awesome for icons. The background particles are created with Particle.js library.

Interactive Features

Click on the heart, letters, or buttons to create stunning visual effects. The heart beats with animation, letters can be swapped, and colors can be changed dynamically.

Design Philosophy

The design represents connection and love through visual elements. The "F" and "R" letters symbolize two entities connected through a pulsating heart in the center.
